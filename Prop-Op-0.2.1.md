Prop : Op : 0.2.1 : $(f(f) \land (f \neq f)) \Rightarrow (f(f) \neq f)$, If $f$ represents itself as something $f$ is not, $f$ is not what $f$ represents itself as.
```
(=>
	(!= (f f) f)
	(= (f f) (~f))
)
```
